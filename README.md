 # Retail Sales Dashboard
This Power BI dashboard provides insights into retail sales performance.

## Features
- Total Sales, Profit, and Growth %
- Sales trends by month/quarter
- Top product categories and regions
- Customer segmentation analysis
- Interactive filters for date, region, and category

## Files
- `Retail_Sales.pbix` → Power BI report
- `retail_dataset.csv` → Sample dataset
- `screenshots/` → Dashboard preview images

## Tools Used:
 Power BI | DAX | Power Query
